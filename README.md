# Drush Reload Parallel

Reloads Drupal databases using [drush](https://github.com/drush-ops/drush).

## Installation

```bash
$ npm install -g drush-reloadp
```

## Usage

```bash
$ drush-reloadp -s @source.alias -d @dest.alias
```
